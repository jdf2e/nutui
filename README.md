## ⌨️ 本地开发

你可以使用 Gitpod 进行在线开发：

[![Open in Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://gitpod.io/#https://github.com/jdf2e/nutui.git)

或者克隆到本地开发:

```bash
$ git clone https://github.com/jdf2e/nutui.git
$ cd nutui
$ npm install
$ npm run dev
```
